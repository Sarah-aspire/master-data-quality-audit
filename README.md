# Master Data Quality Audit

Practical master data quality audit project focused on data validation, reporting, and process-oriented data quality improvement workflows.

## Project Overview

This project demonstrates how structured validation and reporting processes can improve the quality, consistency, and reliability of customer master data.

The workflow includes:

* Data quality checks
* Duplicate detection
* Missing value analysis
* Data standardization
* Validation reporting
* KPI-based monitoring

## Technologies

* Python (Pandas)
* SQL
* Excel / CSV
* Power BI
* ## Key Validation Checks

* Duplicate service code detection

* Cost center format validation
* Data consistency checks
* KPI-based quality monitoring
